# Parkinson Disease Detection
A ML model for accurately detecting Parkinson Disease in patients and help them recover at an early stage in their lives.

<hr>

## <b>Tech Stack used-:</b>

### <u>Front End</u> 
* StreamLit
* HTML
* CSS
* Python

### <u>Machine Learning</u>
* Python
* Scikit-Learn
* Numpy
* Pandas
* XGBoost

### <u>Dataset used</u>
* UCI Parkinson's dataset
